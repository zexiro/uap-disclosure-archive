## How to attribute when reusing

If you republish parts of this archive:

1. Credit the originating agency for any specific document (e.g. *"FBI case
   file 62-HQ-83894, released by the FBI under FOIA, mirrored from
   war.gov/UFO Release 1 (May 2026)."*).
2. Note that materials are public-domain US government works under § 105.
3. Don't imply endorsement by the DoD, FBI, NASA, State Department, or any
   US government agency.
4. The **search UI**, **vault structure**, **relationship-link algorithm**,
   and any **derived analysis** are this project's authored work — credit
   this repository if you reuse those.

## OCR caveat

Text extracted from scanned PDFs (FBI files in particular) was produced by
[Tesseract](https://github.com/tesseract-ocr/tesseract) via
[ocrmypdf](https://github.com/ocrmypdf/OCRmyPDF). OCR is imperfect; quote the
original PDF, not the OCR sidecar, when accuracy matters.

## Privacy / takedown

If a record contains personally identifiable information that should have
been redacted by the originating agency, please open an issue. The mirror
will preserve a note of the takedown for the public record while removing
the offending material.
